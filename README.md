# actout

actout is a Monte Carlo simulator for actin filament dynamics.  It is
implemented efficiently, using the Gillespie algorithm.

## Installation

### Dependencies

- CMake
- Google Protocol Buffers
- Google Test

### Building the project
mkdir build
cd build
cmake ..
make
make test
